# **App Name**: FinTrack

## Core Features:

- Expense Tracking: Log and categorize income and expenses.
- Budgeting Tool: Set monthly budgets for each category and track progress.
- Spending Visualization: Visualize spending patterns with interactive charts.
- Financial Insights: AI powered tool that suggests areas to cut costs based on spending patterns.
- Dashboard: A clean dashboard with key insights and summaries of financial data.
- Custom Categories: Users can define their own categories for income and expenses to facilitate granular tracking.

## Style Guidelines:

- Primary color: Deep blue (#293462) to evoke trust and security.
- Background color: Light gray (#F0F4F8) for a clean, unobtrusive backdrop.
- Accent color: Teal (#00A3AD) to highlight key data points and actions.
- Body and headline font: 'Inter', a grotesque-style sans-serif.
- Use flat, minimalist icons for categories and actions.
- Clean, intuitive layout with a focus on data visualization.
- Subtle transitions and animations to enhance user experience.
